### GIT-02: App shows last commit message per project

**Phase:** 2 (Data Layer)
**Root cause:** `getGitData()` in `git-reader.ts` correctly fetches `lastCommitMessage`. The data flows through `scanner.ts` → `page.tsx` → `CodeProjectCard` props. However, `CodeProjectCard` only renders `lastCommitTimestamp` (as relative time) — it never displays `lastCommitMessage`.

**Fix:** Add a render line in `codeProjectCard.tsx` to display `project.lastCommitMessage` (e.g., under the timestamp row). This is a one-line UI addition.
