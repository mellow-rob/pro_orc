<task type="auto">
  <name>Task 2: Configure n3urala1 dark theme in globals.css</name>
  <files>
    pro-orc/app/globals.css
  </files>
  <action>
    Working directory: /Users/rob/project_orchestration/pro-orc/

    Replace the contents of app/globals.css entirely with the n3urala1 dark theme CSS.
    This overrides whatever shadcn init generated. The structure below is correct for Tailwind v4 + shadcn.

    CRITICAL requirements for globals.css:
    1. Must start with `@import "tailwindcss"` (NOT @tailwind base/components/utilities)
    2. Must have `@import "tw-animate-css"` (NOT tailwindcss-animate)
    3. Must have `@custom-variant dark (&:is(.dark *))` — makes dark: utilities respond to class, not prefers-color-scheme
    4. Must have `@theme inline { ... }` mapping CSS vars to Tailwind color utilities
    5. .dark block must use OKLCH values for all colors
    6. Must have @layer utilities with glow-cyan and glow-fuchsia custom classes

    Write the full globals.css:

    ```css
    @import "tailwindcss";
    @import "tw-animate-css";

    /* Dark variant fires when any ancestor has .dark class.
       Without this, dark: utilities use prefers-color-scheme (system).
       Since we hardcode class="dark" on <html>, this line is REQUIRED. */
    @custom-variant dark (&:is(.dark *));

    @theme inline {
      /* Font families — mapped from next/font CSS variables */
      --font-sans: var(--font-inter);
      --font-mono: var(--font-mono);

      /* Color mappings — Tailwind bg-background, text-foreground etc. use these */
      --color-background: var(--background);
      --color-foreground: var(--foreground);
      --color-card: var(--card);
      --color-card-foreground: var(--card-foreground);
      --color-border: var(--border);
      --color-input: var(--input);
      --color-ring: var(--ring);
      --color-primary: var(--primary);
      --color-primary-foreground: var(--primary-foreground);
      --color-secondary: var(--secondary);
      --color-secondary-foreground: var(--secondary-foreground);
      --color-muted: var(--muted);
      --color-muted-foreground: var(--muted-foreground);
      --color-accent: var(--accent);
      --color-accent-foreground: var(--accent-foreground);
      --color-destructive: var(--destructive);

      /* Border radius scale */
      --radius-sm: calc(var(--radius) - 4px);
      --radius-md: calc(var(--radius) - 2px);
      --radius-lg: var(--radius);
      --radius-xl: calc(var(--radius) + 4px);
    }

    /* ============================================================
       Light mode defaults (not used in v1 — dark is hardcoded)
       Kept for completeness; .dark block below overrides everything.
       ============================================================ */
    :root {
      --radius: 0.5rem;
      --background: oklch(0.985 0 0);
      --foreground: oklch(0.145 0 0);
      --card: oklch(0.985 0 0);
      --card-foreground: oklch(0.145 0 0);
      --primary: oklch(0.205 0 0);
      --primary-foreground: oklch(0.985 0 0);
      --secondary: oklch(0.97 0 0);
      --secondary-foreground: oklch(0.205 0 0);
      --muted: oklch(0.97 0 0);
      --muted-foreground: oklch(0.556 0 0);
      --accent: oklch(0.97 0 0);
      --accent-foreground: oklch(0.205 0 0);
      --destructive: oklch(0.577 0.245 27.325);
      --border: oklch(0.922 0 0);
      --input: oklch(0.922 0 0);
      --ring: oklch(0.708 0 0);
    }

    /* ============================================================
       n3urala1 Dark Theme — THE active theme.
       class="dark" is hardcoded on <html> in layout.tsx.
       Palette: Dark Navy bg / Cyan primary / Fuchsia accent
       Reference: n3urala1.com aesthetic
       ============================================================ */
    .dark {
      /* Background: ultra-dark navy (#0a0f1a approximate) */
      --background: oklch(0.11 0.02 264);
      --foreground: oklch(0.95 0 0);

      /* Cards: semi-transparent glassmorphism over dark background
         Equivalent to rgba(255,255,255,0.03) — subtle, readable */
      --card: oklch(0.18 0.015 264 / 0.6);
      --card-foreground: oklch(0.92 0 0);

      /* Primary: Cyan (#06b6d4) — code projects, actions, CTAs */
      --primary: oklch(0.715 0.143 212.34);
      --primary-foreground: oklch(0.11 0.02 264);

      /* Secondary: subtle surface for secondary elements */
      --secondary: oklch(0.22 0.015 264);
      --secondary-foreground: oklch(0.85 0 0);

      /* Muted: subdued backgrounds and text */
      --muted: oklch(0.18 0.01 264);
      --muted-foreground: oklch(0.62 0.01 264);

      /* Accent: Fuchsia (#d946ef) — research projects, highlights */
      --accent: oklch(0.70 0.22 320.08);
      --accent-foreground: oklch(0.11 0.02 264);

      /* Destructive: red for errors */
      --destructive: oklch(0.577 0.245 27.325);

      /* Borders: very subtle white — border-white/5 equivalent */
      --border: oklch(1 0 0 / 0.05);
      --input: oklch(1 0 0 / 0.05);

      /* Ring: Cyan for focus rings */
      --ring: oklch(0.715 0.143 212.34);
    }

    /* ============================================================
       Custom utilities not covered by shadcn defaults
       ============================================================ */
    @layer utilities {
      /* Cyan glow — for code project cards and primary actions */
      .glow-cyan {
        box-shadow: 0 0 20px oklch(0.715 0.143 212.34 / 0.15);
      }

      /* Fuchsia glow — for research project cards and highlights */
      .glow-fuchsia {
        box-shadow: 0 0 20px oklch(0.70 0.22 320.08 / 0.15);
      }

      /* Atmospheric blur orbs (n3urala1 background depth effect) */
      .bg-orb-cyan {
        background: oklch(0.715 0.143 212.34 / 0.08);
        filter: blur(100px);
      }

      .bg-orb-fuchsia {
        background: oklch(0.70 0.22 320.08 / 0.08);
        filter: blur(100px);
      }
    }
    ```

    After writing the file, also check the shadcn-generated component files (card.tsx, button.tsx, badge.tsx) for Tailwind v3 class names that don't work in v4. Apply these replacements in any component file that contains them:
    - `shadow-sm` → `shadow-xs`
    - `shadow` (standalone) → `shadow-sm`
    - `rounded-sm` → `rounded-xs`
    - `ring` (standalone) → `ring-3`
    - `outline-none` → `outline-hidden`
    - `bg-opacity-*` → use `/` syntax (e.g., `bg-white/50`)
    - `flex-shrink-0` → `shrink-0`
    - `flex-grow` → `grow`
