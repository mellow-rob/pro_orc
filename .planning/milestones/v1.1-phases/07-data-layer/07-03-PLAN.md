  <behavior>
**readGitData(String projectPath, {String gitBinary = 'git'}) -> Future&lt;GitData&gt;**

1. Run `git log --format=%H%n%aI%n%s -1` in projectPath with `runInShell: true`
   - Parse stdout: line 1 = full hash (truncate to 7 chars), line 2 = ISO date, line 3+ = subject
   - Wrap with `Future.any([processRun, Future.delayed(5s, throw TimeoutException)])` for timeout
2. Run `git remote get-url origin` in projectPath with `runInShell: true`
   - Parse stdout: normalize SSH (`git@github.com:owner/repo.git` -> `https://github.com/owner/repo`) and HTTPS (`https://github.com/owner/repo.git` -> strip .git)
   - Same 5-second timeout wrapper
3. On any error (not a git repo, timeout, no remote, exitCode != 0) -> return GitData.empty

**readAllGitData(List&lt;String&gt; projectPaths, {String gitBinary = 'git'}) -> Future&lt;List&lt;GitData&gt;&gt;**

- Chunk projectPaths into groups of 5
- For each chunk: `Future.wait(chunk.map((path) => readGitData(path, gitBinary: gitBinary)))` with individual `.catchError((_) => GitData.empty)`
- Collect all results in order

**Cases:**
- Valid git repo with remote → full GitData
- Valid git repo, no remote → GitData with commit info but null githubUrl
- Not a git repo → GitData.empty
- Nonexistent directory → GitData.empty
- SSH remote `git@github.com:owner/repo.git` → `https://github.com/owner/repo`
- HTTPS remote `https://github.com/owner/repo.git` → `https://github.com/owner/repo`
- Non-GitHub remote → githubUrl = null
- Timeout → GitData.empty
- readAllGitData with 12 paths → 3 chunks (5, 5, 2), all results returned in order
  </behavior>
