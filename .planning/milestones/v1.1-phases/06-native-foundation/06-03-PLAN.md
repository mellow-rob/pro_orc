  <action>
    1. Build the macOS release binary:
       ```bash
       cd ~/project_orchestration/pro_orc
       flutter build macos
       ```
       Must succeed without errors.

    2. Verify sandbox is disabled in the built binary's entitlements:
       ```bash
       codesign -d --entitlements - build/macos/Build/Products/Release/pro_orc.app 2>&1 | grep -A1 "app-sandbox"
       ```
       Must show `<false/>`.

    3. Verify the built .app can read from the filesystem (sandbox validation):
       ```bash
       # Launch the app briefly to verify it starts
       open build/macos/Build/Products/Release/pro_orc.app
       sleep 3
       # The app should be running — check process
       pgrep -f "pro_orc" && echo "App is running"
       ```

    4. Verify no Dock icon (LSUIElement check):
       ```bash
       # Check the built Info.plist
       defaults read build/macos/Build/Products/Release/pro_orc.app/Contents/Info.plist LSUIElement
       ```
       Must return `1`.

    5. Cross-check both source entitlement files:
       ```bash
       echo "=== DebugProfile ===" && grep "app-sandbox" macos/Runner/Runner-DebugProfile.entitlements
       echo "=== Release ===" && grep "app-sandbox" macos/Runner/Runner-Release.entitlements
       ```
       Both must show `<false/>`.

    6. Kill the test app:
       ```bash
       pkill -f "pro_orc" 2>/dev/null || true
       ```
  </action>
  <verify>
    All five checks pass:
    1. `flutter build macos` exits 0
    2. Built binary entitlements show sandbox = false
    3. App process starts successfully
    4. LSUIElement = 1 in built Info.plist
    5. Both source entitlement files show sandbox = false
  </verify>
