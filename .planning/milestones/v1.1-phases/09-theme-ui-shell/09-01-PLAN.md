```
// Background layers
bgBase:   Color(0xFF0A0A0F)    // oklch(0.065 0.015 260)
bgSurf:   Color(0xFF0A1017)    // oklch(0.17  0.018 255)
bgElev:   Color(0xFF11161E)    // oklch(0.20  0.018 255)
bgCard:   Color(0xFF161B22)    // oklch(0.22  0.016 255)

// Cyan (PRIMARY)
cyanHi:   Color(0xFF00DEEB)    // oklch(0.80  0.18  200)
cyan:     Color(0xFF00CDDC)    // oklch(0.74  0.20  200)
cyanLo:   Color(0xFF009FAF)    // oklch(0.60  0.20  200)
cyanOrb:  Color(0xFF00879A)    // oklch(0.52  0.22  200)

// Fuchsia (SECONDARY)
fuchHi:   Color(0xFFFA48FA)    // oklch(0.72  0.28  328)
fuch:     Color(0xFFD710D8)    // oklch(0.62  0.28  328)
fuchLo:   Color(0xFFAF00B1)    // oklch(0.52  0.26  328)
fuchOrb:  Color(0xFFA600A9)    // oklch(0.48  0.28  328)

// Text (gedaempft — not pure white)
textPri:  Color(0xFFE0E5EB)    // oklch(0.92  0.01  255) — ~#E0E0E8
textSec:  Color(0xFF9399A0)    // oklch(0.68  0.012 255)
textDim:  Color(0xFF5F6469)    // oklch(0.50  0.010 255)
textDis:  Color(0xFF404347)    // oklch(0.38  0.008 255)
```

Implement `copyWith()` with all fields. Implement `lerp()` using `Color.lerp` for all fields.

Rewrite `glow_border_shell.dart` to:

3. **Remove `Border.all(...)`** entirely from the BoxDecoration — locked decision: no border on cards/shells.
4. Replace ALL `withOpacity(x)` calls with `withValues(alpha: x)` — `withOpacity` is deprecated in Flutter 3.38+.
5. Keep the existing `ClipRRect` structure (needed for orb background clipping in Plan 02).
6. Keep the outer glow `BoxShadow` effects but use AppColors tokens:
   - Cyan glow: `colors.cyan.withValues(alpha: 0.15)`, blurRadius 20, spreadRadius 2
   - Fuchsia glow: `colors.fuch.withValues(alpha: 0.08)`, blurRadius 40, spreadRadius 0
7. Background color: `colors.bgBase`

The result: A glowing container shell with no border, correct tokens, no deprecated APIs.
