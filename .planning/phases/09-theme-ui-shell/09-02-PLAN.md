**Create `lib/features/shell/orb_background.dart`:**

A `StatefulWidget` with `TickerProviderStateMixin` that paints 3 animated orbs on a `CustomPainter` canvas.

3 `AnimationController` instances with different durations for visual stagger (no explicit offset chaining — different periods create natural desync):
- Controller 1: 18 seconds, drives cyan orb top-left area
- Controller 2: 23 seconds, drives fuchsia orb bottom-right area
- Controller 3: 28 seconds, drives cyan orb top-right area (subtler)

All controllers: `.repeat(reverse: true)` — continuous slow drift.

`_OrbPainter extends CustomPainter`:
- Constructor takes 3 `Animation<double>` controllers. Uses `super(repaint: Listenable.merge([c1, c2, c3]))` for efficient repainting.
- `shouldRepaint` returns `false` (repaint driven by Listenable.merge).
- `paint()` draws 3 radial gradient circles:

  Orb 1 (Cyan, top-left drift):
  - center: `Offset(width * (0.15 + 0.20 * c1.value), height * (0.20 + 0.15 * c1.value))`
  - radius: `shortestSide * 0.35`
  - color: `AppColors.dark.cyanOrb` (Color(0xFF00879A))
  - opacity: 0.18

  Orb 2 (Fuchsia, bottom-right drift):
  - center: `Offset(width * (0.70 + 0.18 * c2.value), height * (0.65 + 0.18 * c2.value))`
  - radius: `shortestSide * 0.40`
  - color: `AppColors.dark.fuchOrb` (Color(0xFFA600A9))
  - opacity: 0.14

  Orb 3 (Cyan, top-right, slower):
  - center: `Offset(width * (0.75 - 0.15 * c3.value), height * (0.15 + 0.12 * c3.value))`
  - radius: `shortestSide * 0.28`
  - color: `AppColors.dark.cyanOrb` (Color(0xFF00879A))
  - opacity: 0.10

Each orb drawn with `Paint()..shader = RadialGradient(colors: [color.withValues(alpha: opacity), color.withValues(alpha: 0.0)], stops: [0.0, 1.0]).createShader(Rect.fromCircle(center: center, radius: radius))`.

`build()` method: wrap `CustomPaint(painter: _OrbPainter(...))` in a `RepaintBoundary` for performance isolation. The widget takes no `child` — it's used as a `Positioned.fill` in a Stack.

Dispose all 3 controllers in `dispose()`.

**Create `lib/features/shell/glass_card.dart`:**

A `StatelessWidget` with parameters:
- `required Widget child`
- `double borderRadius = 14.0`
- `double blurSigma = 12.0`

Build method:
```
ClipRRect(
  borderRadius: BorderRadius.circular(borderRadius),
  child: BackdropFilter(
    blendMode: BlendMode.src,  // mitigates white halo on dark backgrounds
    filter: ImageFilter.blur(sigmaX: blurSigma, sigmaY: blurSigma),
    child: Container(
      decoration: BoxDecoration(
        color: colors.bgCard.withValues(alpha: 0.55),  // glass tint
        borderRadius: BorderRadius.circular(borderRadius),
        // NO border — locked decision
      ),
      child: child,
    ),
  ),
)
```

Read `colors` from `Theme.of(context).extension<AppColors>()!`. Import `dart:ui` for `ImageFilter`.

IMPORTANT: Use `withValues(alpha:)` everywhere, never `withOpacity()`.

Add `int _selectedIndex = 0;` state field.

New build method structure (following research pattern — orbs OUTSIDE scaffold):

```dart
final colors = Theme.of(context).extension<AppColors>()!;

return GlowBorderShell(
  child: Stack(
    children: [
      const Positioned.fill(child: OrbBackground()),
      Scaffold(
        backgroundColor: Colors.transparent,  // let orbs show through
        body: Padding(
          padding: const EdgeInsets.only(top: 30),  // existing titlebar clearance
          child: Row(
            children: [
              NavigationRail(
                backgroundColor: Colors.transparent,
                selectedIndex: _selectedIndex,
                onDestinationSelected: (i) => setState(() => _selectedIndex = i),
                labelType: NavigationRailLabelType.selected,
                minWidth: 80,
                destinations: const [
                  NavigationRailDestination(
                    icon: Icon(Icons.code_outlined),
                    selectedIcon: Icon(Icons.code),
                    label: Text('Code'),
                  ),
                  NavigationRailDestination(
                    icon: Icon(Icons.science_outlined),
                    selectedIcon: Icon(Icons.science),
                    label: Text('Research'),
                  ),
                  NavigationRailDestination(
                    icon: Icon(Icons.smart_toy_outlined),
                    selectedIcon: Icon(Icons.smart_toy),
                    label: Text('Claude Tools'),
                  ),
                ],
              ),
              Expanded(
                child: IndexedStack(
                  index: _selectedIndex,
                  children: const [
                    CodeTab(),
                    ResearchTab(),
                    ClaudeToolsTab(),
                  ],
                ),
              ),
            ],
          ),
        ),
      ),
    ],
  ),
);
```

Key points:
- `OrbBackground` is `Positioned.fill` in a `Stack` BEHIND the `Scaffold` — orbs bleed behind NavigationRail and all tab content.
- `Scaffold.backgroundColor = Colors.transparent` so orbs show through.
- `NavigationRail.backgroundColor = Colors.transparent` — theme-level NavigationRailThemeData (from Plan 01) handles colors.
- No `VerticalDivider` — let the glassmorphism aesthetic define visual separation.
- Keep `ref.watch(projectsProvider)` call in build for reactive watcher chain (even though it's not displayed in tabs yet — it keeps the provider alive). Store in a local variable to suppress unused warnings, or keep the existing switch block as a small status indicator somewhere if appropriate — Claude's discretion.
