```dart
Future<List<ExternalResource>> detectExternalResources(ProjectModel project) async { ... }
```

**Detection logic (in order):**

1. **Notion** (CLN-01): Check `project.gsd?.notionUrl`. If non-null, add ExternalResource with type `notion`, label "Notion-Seite", uri = the URL, hint "Notion-Seite manuell im Browser loeschen".

2. **GitHub** (CLN-02): Check `project.git?.githubUrl`. If non-null, add ExternalResource with type `github`, label "GitHub-Repository", uri = the URL, hint "Repository via `gh repo delete` loeschen oder manuell auf GitHub".

3. **Claude Memory** (CLN-04): Use `encodeProjectPath(project.path)` from memory_reader.dart. Check if directory `~/.claude/projects/{encodedPath}` exists (use `Platform.environment['HOME']`). If it exists, add ExternalResource with type `claudeMemory`, label "Claude Memory", uri = the directory path, hint "Claude-Projektverzeichnis loeschen (Memory + Einstellungen)".

4. **Figma + other URLs** (CLN-03): Scan `.md` files in project root and `.planning/` directory (max 2 levels deep). Use a regex to find URLs: `https?://[^\s)>\]]+`. Filter out:
   - The already-detected Notion URL
   - The already-detected GitHub URL
   - Common non-resource URLs (localhost, 127.0.0.1, example.com)
   - Duplicate URLs

   For each remaining URL:
   - If domain contains `figma.com` → type `figma`, label "Figma-Design"
   - If domain contains `firebase` or `console.firebase.google.com` → type `other`, label "Firebase-Projekt"
   - If domain contains `vercel.com` → type `other`, label "Vercel-Deployment"
   - Otherwise → type `other`, label = domain name (e.g. "linear.app")

   Hint for all URL-based resources: "Manuell im Browser oeffnen und ggf. loeschen"

   Limit to max 10 URLs to avoid noise. Skip files larger than 100KB (binary noise).

**Error handling:** Wrap everything in try/catch, return empty list on any error. Individual detection steps should not abort the whole function — use separate try/catch for each step.

Import `package:path/path.dart` as `p` for path operations. Import `encodeProjectPath` from memory_reader.dart. Import ProjectModel and ExternalResource.
