    **claude_tools_scanner.dart** ‚Äî Create pure Dart service (no Flutter imports), mirrors ProjectScanner pattern:

    - `ClaudeToolsScanner` class with `final String claudeDir` field.
    - Constructor: `ClaudeToolsScanner({String? claudeDirOverride})` ‚Äî default: `'${Platform.environment['HOME']}/.claude'`. The override param enables testing with temp dirs.
    - `Future<ClaudeToolsData> scanAll()` ‚Äî calls three private methods, catches top-level errors, returns `ClaudeToolsData` (with `hasError: true` on failure).

    **_scanSkills():**
    - List `$claudeDir/skills/` subdirectories
    - For each subdir, look for `SKILL.md` then `skill.md` (case-insensitive fallback per research)
    - If found, parse YAML frontmatter: extract `name`, `description`, `homepage` using hand-rolled parser (split at `---` delimiters, split each line at first `:`, strip surrounding quotes)
    - If no SKILL.md found: use folder name as display name, no description (per Claude's Discretion decision)
    - Sort alphabetically by name
    - Return `List<SkillData>`

    **_scanPlugins():**
    - Read `$claudeDir/plugins/installed_plugins.json` (jsonDecode)
    - Read `$claudeDir/settings.json` for `enabledPlugins` map
    - Read `$claudeDir/plugins/known_marketplaces.json` for marketplace URLs (optional, catch if missing)
    - For each key in `plugins` map: take `entries[0]` (first install), split key at `@` for name/marketplace
    - Read description from `installPath/.claude-plugin/plugin.json` if available
    - Derive marketplace URL from `known_marketplaces.json` ‚Üí `source.repo` ‚Üí `https://github.com/$repo`
    - Sort alphabetically by name
    - Return `List<PluginData>`

    **_scanMcpServers():**
    - Read `$claudeDir/settings.json` ‚Üí `mcpServers` (Map)
    - Handle empty `{}` gracefully (return empty list)
    - For each entry: determine type from `config['type']` field (default: stdio). For stdio: combine command + args. For http/sse: use URL.
    - Sort alphabetically by name
    - Return `List<McpServerData>`

    **_parseFrontmatter(String content):**
    - Private helper method returning `Map<String, String>`
    - Find content between first two `---` lines
    - For each line, split at first `:` only (handles colons in quoted values)
    - Strip surrounding double quotes from values
    - Return key-value map

    All file I/O wrapped in try/catch ‚Äî return empty lists on errors, never throw. Use `dart:io` Directory.list and File.readAsString (async). Use package imports only.
