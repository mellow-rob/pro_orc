    **1. Create ProjectDetailPanel** (`features/shared/project_detail_panel.dart`):
    Stateless widget (or ConsumerWidget) accepting `ProjectModel project`.

    Opens via `showGeneralDialog` with slide-up + fade animation (Claude's Discretion):
    ```dart
    Future<void> showProjectDetail(BuildContext context, ProjectModel project) async {
      await showGeneralDialog(
        context: context,
        barrierDismissible: true,
        barrierLabel: 'Close',
        barrierColor: Colors.black54,
        transitionDuration: const Duration(milliseconds: 300),
        transitionBuilder: (context, animation, secondaryAnimation, child) {
          return SlideTransition(
            position: Tween<Offset>(
              begin: const Offset(0, 0.05),
              end: Offset.zero,
            ).animate(CurvedAnimation(
              parent: animation,
              curve: Curves.easeOutCubic,
            )),
            child: FadeTransition(opacity: animation, child: child),
          );
        },
        pageBuilder: (context, animation, secondaryAnimation) =>
            ProjectDetailPanel(project: project),
      );
    }
    ```

    **Panel layout:** A Center widget containing a ConstrainedBox (maxWidth: 600, maxHeight: 80% of screen) wrapping a GlassCard with scrollable content.

    **Content sections (SingleChildScrollView > Column):**
