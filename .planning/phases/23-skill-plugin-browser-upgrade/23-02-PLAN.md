**Provider changes (claude_tools_provider.dart):**

1. Add a `selectedProjectPathProvider` as a `StateProvider<String?>` (null = global view, String = project path):
```dart
final selectedProjectPathProvider = StateProvider<String?>((ref) => null);
```

2. Add a `projectToolsProvider` that scans per-project tools when a project is selected:
```dart
final projectToolsProvider = FutureProvider<ClaudeToolsData?>((ref) async {
  final projectPath = ref.watch(selectedProjectPathProvider);
  if (projectPath == null) return null;
  return ClaudeToolsScanner().scanProjectTools(projectPath);
});
```

Keep the existing `claudeToolsProvider` unchanged — it always provides global tools.

**Tab changes (claude_tools_tab.dart):**

1. Import `projectsProvider` and the new `selectedProjectPathProvider`/`projectToolsProvider`.

2. Between the search field and the sections, add a project selector dropdown:
   - Watch `projectsProvider` for the project list
   - Watch `selectedProjectPathProvider` for current selection
   - Build a `DropdownButton<String?>` styled with the glassmorphism theme:
     - First item: `"Alle Tools (Global)"` with value `null`
     - Then each project: `project.displayName` with value `project.path`
   - On change: `ref.read(selectedProjectPathProvider.notifier).state = value`
   - Style the dropdown to match the dark theme: `dropdownColor: colors.bgElev`, text color `colors.textPri`, icon color `colors.textSec`

3. When a project is selected (not null):
   - Watch `projectToolsProvider` for per-project tools
   - Merge global + project tools for display: show global tools as-is, then add a "Projekt-spezifisch" subsection under Skills and MCP-Server showing project-scoped items
   - Plugins section shows all plugins (always global) with a subtle "Global" scope label
   - Skills section: global skills first, then project skills with a `"Projekt"` scope badge
   - MCP section: global servers first, then project servers with a `"Projekt"` scope badge

4. The scope badge is a small chip similar to the Aktiv/Inaktiv badge: `Container` with `padding: EdgeInsets.symmetric(horizontal: 6, vertical: 2)`, background `colors.cyanLo.withValues(alpha: 0.15)`, text `"Projekt"` in `colors.cyanLo` at fontSize 10.

5. When no project is selected (null / default), show the existing global-only view unchanged.
