Create `pro-orc/lib/tools-scanner.ts` following the `scanner.ts` pattern (`import 'server-only'`, `fs.promises`, `Promise.all` for concurrency, null-safe reads).

**Structure:**

1. `import 'server-only'` at top
2. Import `{ promises as fs }` from `fs`, `path` from `path`, `{ PATHS }` from `@/lib/paths`
3. Import `ClaudeTool, ClaudeToolsData` from `@/lib/types`

**Functions to implement:**

**`readSkillMd(dirPath: string)`** — reads `skill.md` (try lowercase first, then `SKILL.md`). Splits on `---` delimiters, parses YAML frontmatter with `js-yaml` (`import yaml from 'js-yaml'`). Returns `{ name?: string; description?: string } | null`. Wraps each file read attempt in try/catch.

**`scanSkills()`** — reads `path.join(PATHS.claude, 'skills')` with `fs.readdir(dir, { withFileTypes: true })`. Filters for `isDirectory() || isSymbolicLink()`. For each entry: resolve symlinks via `fs.realpath()`, call `readSkillMd()`, return `ClaudeTool` with `type: 'skill'`. If skills dir missing, return `[]`. Use `Promise.all` for concurrent reads.

**`readSettings()`** — reads `path.join(PATHS.claude, 'settings.json')`, parses JSON, returns `settings.enabledPlugins ?? {}` as `Record<string, boolean>`. Returns `{}` on any error.

**`scanPlugins(enabledPlugins: Record<string, boolean>)`** — reads `path.join(PATHS.claude, 'plugins', 'installed_plugins.json')`, parses JSON. For each entry in `registry.plugins`:
  - Split key on `@` to get `[pluginName, marketplace]`
  - Take `installs[0]` (first/only entry)
  - Read `.claude-plugin/plugin.json` from `install.installPath` for `name` and `description`
  - Detect MCP: `fs.access(path.join(install.installPath, '.mcp.json')).then(() => true).catch(() => false)`
  - Set `type: isMcp ? 'mcp' : 'plugin'`
  - Set `enabled: enabledPlugins[key] ?? false`
  - Use `Promise.all` for concurrent processing

**`scanClaudeTools()`** (exported) — reads settings, then runs `scanSkills()` and `scanPlugins(enabledPlugins)` concurrently via `Promise.all`. Splits plugins into `mcpPlugins` and `skillPlugins` by `type`. Returns `ClaudeToolsData` with `scannedAt: new Date().toISOString()`.

**Critical implementation details (from research):**
- Symlinks MUST be resolved via `fs.realpath()` before reading skill.md
- skill.md filename is case-insensitive — try both `skill.md` and `SKILL.md`
- `installed_plugins.json` values are arrays — always access `[0]`
- All file reads wrapped in try/catch returning fallback (null, `{}`, or `[]`)
- Use `js-yaml` (already installed) for YAML parsing, not gray-matter (not installed)
