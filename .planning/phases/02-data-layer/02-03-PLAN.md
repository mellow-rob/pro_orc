  <action>
Create `lib/scanner.ts` with `import 'server-only'` at the top.

Import:
- `promises as fs` from `fs`
- `path` from `path`
- `PATHS, projectIdFromPath` from `@/lib/paths`
- `Project, CodeProject, ResearchProject` from `@/lib/types`
- `parseGsdData` from `@/lib/parser`
- `getGitData` from `@/lib/git-reader`

Internal helper `scanDir(rootPath: string, type: 'code' | 'research')`:
1. `await fs.readdir(rootPath, { withFileTypes: true })`
2. Filter: `entry.isDirectory() && !entry.name.startsWith('.')` — excludes .git, .venv, etc.
3. For each directory entry:
   - `projectPath = path.join(rootPath, entry.name)`
   - `id = projectIdFromPath(projectPath)`
   - `name = entry.name` — raw directory name as display name (NOT slugified id)
   - `gsdData = await parseGsdData(projectPath)`
4. Return array of base project objects (without git data yet)

Export `async function scanProjects(): Promise<Project[]>`:
1. Scan both roots concurrently:
   ```typescript
   const [codeDirs, researchDirs] = await Promise.all([
     scanDir(PATHS.code, 'code').catch(() => []),
     scanDir(PATHS.research, 'research').catch(() => []),
   ])
   ```
   The `.catch(() => [])` handles missing root directories gracefully.

2. For code projects, enrich with git data using **Promise.allSettled** (GIT-03):
   ```typescript
   const gitResults = await Promise.allSettled(
     codeDirs.map(p => getGitData(p.path))
   )
   const codeProjects: CodeProject[] = codeDirs.map((project, i) => {
     const gitResult = gitResults[i]
     const gitData = gitResult.status === 'fulfilled' ? gitResult.value : {}
     return { ...project, type: 'code' as const, ...gitData }
   })
   ```

3. Research projects get no git data:
   ```typescript
   const researchProjects: ResearchProject[] = researchDirs.map(p => ({
     ...p, type: 'research' as const
   }))
   ```

4. Return `[...codeProjects, ...researchProjects]`

**Key design points:**
- `entry.name` is the display name (e.g., "landlord_checker_gsd"), NOT the slugified id
- Type discrimination comes from WHICH root dir, not from .git presence
- Git calls are concurrent (Promise.allSettled), not sequential
- Missing scan roots return empty arrays, not errors
