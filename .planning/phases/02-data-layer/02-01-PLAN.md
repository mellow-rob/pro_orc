  <action>
Create `lib/parser.ts` with `import 'server-only'` at the top.

Export interface `GsdParseResult`:
```typescript
export interface GsdParseResult {
  gsdStatus?: GsdStatus
  currentPhase?: string
  nextStep?: string
  phaseProgress?: number
  notionUrl?: string
}
```

Export `async function parseGsdData(projectPath: string): Promise<GsdParseResult>`.

Internal helper `readFile(filePath: string): Promise<string | null>` — try/catch around `fs.readFile`, returns null on any error (ENOENT, EACCES, mid-save corruption).

Use `planningDir(projectPath)` from `@/lib/paths` to resolve file paths.

Read all three files concurrently with `Promise.all([readFile(state), readFile(roadmap), readFile(project)])`.

If ALL three are null, return `{}` — this is "no GSD data", not an error (SCAN-07).

**parseState(content: string | null)** — returns `{ gsdStatus?, currentPhase?, nextStep? }`:
- currentPhase: try these regex patterns in order, take first match:
  - `/^\*\*Phase:\*\*\s*(.+)$/m`
  - `/^\*\*Current Phase:\*\*\s*(.+)$/m`
  - `/^\*\*Aktuelle Phase:\*\*\s*(.+)$/m`
- gsdStatus: match `/^\*\*Status:\*\*\s*(.+)$/m`, then derive:
  - Contains "complete" (case-insensitive) -> `'done'`
  - Contains "archived" -> `'archived'`
  - Contains "paused" -> `'paused'`
  - Contains "research" -> `'research'`
  - Contains "planning" -> `'planning'`
  - Contains "progress" or any phase mentioned -> `'building'`
  - Default: `undefined`
- nextStep: try these patterns:
  - `/^\*\*Next Action:\*\*\s*(.+)$/m`
  - `/^\*\*Next Step:\*\*\s*(.+)$/m`
  - `/^\*\*Nächster Schritt:\*\*\s*(.+)$/m`

**parseRoadmap(content: string | null)** — returns `{ phaseProgress? }`:
- Count `- [x]` matches (case-insensitive `/^- \[x\]/gim`)
- Count `- [ ]` matches (`/^- \[ \]/gm`)
- total = completed + pending
- If total === 0, return `{}` (no progress data, not 0%)
- Otherwise: `phaseProgress: Math.round((completed / total) * 100)`

**parseProject(content: string | null)** — returns `{ notionUrl? }`:
- Match `<!-- notion: URL -->` with `/<!--\s*notion:\s*(https?:\/\/[^\s>]+)\s*-->/`
- Return `{ notionUrl: match?.[1] }` or `{}` if no match

Every internal function returns partial object — never throws (SCAN-08).
