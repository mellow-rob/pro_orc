  <action>
Create `lib/git-reader.ts` with `import 'server-only'` at the top.

Import `simpleGit` from `simple-git` and `CodeProject` from `@/lib/types`.

Export type `GitFields`:
```typescript
export type GitFields = Pick<CodeProject, 'lastCommitMessage' | 'lastCommitTimestamp' | 'lastCommitSha'>
```

Export `async function getGitData(projectPath: string): Promise<GitFields>`.

Implementation:
1. Create a `simpleGit` instance per call with:
   ```typescript
   const git = simpleGit({
     baseDir: projectPath,
     timeout: {
       block: 5000,    // 5-second hard timeout (GIT-04)
       stdOut: false,   // Do NOT reset timer on stdout — absolute timeout
       stdErr: false,   // Do NOT reset timer on stderr — absolute timeout
     },
   })
   ```

2. Inside try block:
   - `const log = await git.log({ maxCount: 1 })`
   - If `!log.latest` return `{}` (repo exists but has no commits)
   - Return `{ lastCommitMessage: log.latest.message, lastCommitTimestamp: log.latest.date, lastCommitSha: log.latest.hash.slice(0, 7) }`

3. Catch block: return `{}` — handles both "not a git repository" (GIT-05) and timeout errors. No logging, no re-throw. These are expected conditions, not errors.

**Do NOT populate `branch` or `isDirty`** — deferred to Phase 3 per research recommendation (avoids doubling git I/O).
