```dart
import 'dart:io';

/// Permanently deletes a project directory from the filesystem.
/// Returns true if deletion succeeded, false if directory not found or deletion failed.
Future<bool> deleteProject(String projectPath) async {
  final dir = Directory(projectPath);
  if (!dir.existsSync()) return false;
  try {
    await dir.delete(recursive: true);
    return true;
  } catch (_) {
    return false;
  }
}
```

This is the `rm -rf` equivalent — permanent, no Papierkorb/Trash. Returns bool for success/failure so the dialog can show feedback. No exceptions thrown — follows the project convention of returning empty/null/false on errors.
