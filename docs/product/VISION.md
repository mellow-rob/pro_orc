# Pro Orc — Vision

> Pro Orc is the **dashboard and launcher** for every Claude Code project —
> built for founders, consultants, and builders who **want to see results,
> not terminals**. Always one menu bar away, never in the way.

## Pillars

- **Status at a glance** — Every project shows instantly where things
  stand, what's running, and what's next — from real project data, updated
  live.
- **Readable, not technical** — Specs, plans, and roadmaps appear as
  polished views — never as raw Markdown or developer internals.
- **One click to action** — Terminal, editor, Finder, Claude session — every
  action starts directly from the card. The dashboard is also the launcher.
